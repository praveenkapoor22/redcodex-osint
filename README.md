# 🔴 RedcodeX OSINT Framework

An interactive Open-Source Intelligence (OSINT) visualization platform built with React and D3.js.

RedcodeX OSINT is designed for cybersecurity researchers, investigators, journalists, analysts, and students who want a structured and interactive way to explore OSINT tools.

---

## 🌐 Live Demo

👉 https://redcodex-osint.vercel.app/

---

## 🚀 Features

- 🧠 Interactive intelligence tree (D3 powered)
- 🔍 Smart search across tools and categories
- 🗂 Categorized OSINT tools
- ⚡ Fast and lightweight (Vite + React)
- 🌙 Cyber-style UI theme
- 🔗 Direct tool access
- 📢 Community-driven suggestions (coming soon)

---

## 🛠 Tech Stack

- React
- D3.js
- Vite
- JavaScript (ES6+)
- CSS3
- Vercel (Deployment)

   - Category
