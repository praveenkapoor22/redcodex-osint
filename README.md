# RedcodeX OSINT Framework

An interactive open-source intelligence (OSINT) visualization platform
built for cybersecurity researchers, investigators, and analysts.

## Features
- Interactive intelligence tree (D3 powered)
- Categorized OSINT tools
- Fast search
- Open-source
- Community-driven
